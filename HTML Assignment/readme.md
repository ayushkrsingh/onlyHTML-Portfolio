# Portfolio Page Assignment

## Objective

Create a **personal portfolio webpage** that introduces you, your background, your skills, and your work. The page should clearly present information about you and demonstrate what you have learned in HTML.

---

## Important Rule

This assignment must be built using **only HTML**.

* Do **not use CSS**
* Do **not use JavaScript**
* Do **not use any frameworks or libraries**

The goal of this assignment is to practice **pure HTML structure and concepts**.

---

## How to Start

1. Use the **Homework Repository** provided.
2. Open the repository in **VS Code**.
3. Delete any **unnecessary files**.
4. Work only inside the **index.html** file to build your portfolio page.
5. Before pushing your project, create a **README.md** file explaining your portfolio.

---

## What Your Portfolio Must Include

### 1. Navigation Bar

At the top of the page, create a **navigation bar** that allows users to move to different sections of your portfolio.

The navigation should include links such as:

* Home
* About
* Skills
* Projects
* Contact

---

### 2. Page Basics

* A proper **page title** visible in the browser tab
* A **favicon (website icon)** for the page

---

### 3. Introduction Section

* Your **full name**
* A **short introduction** about yourself
* A **profile picture**

---

### 4. About Me

Write one or two paragraphs describing:

* Who you are
* What you are learning
* Your interests in technology or web development

---

### 5. Skills

Create a section that lists your skills such as:

* Programming languages
* Tools you know
* Technologies you are learning

---

### 6. Education

Add your education details in a structured format including:

* Year
* Course or degree
* School/college name

---

### 7. Projects

Mention at least **2–3 projects** you have worked on or want to build.

For each project include:

* Project name
* Short description
* A link if available

---

### 8. Social Links

Add links to at least **two social platforms**, such as:

* GitHub
* LinkedIn
* Instagram
* Personal blog

---

### 9. Contact Section

Include a small contact area where someone could send you a message.

It should collect details like:

* Name
* Email
* Message

---

### 10. Footer

At the bottom of the page include:

* Your name
* The year
* A short message like “Made by [Your Name]”

---

## README Requirement

Before pushing your project to GitHub, add a **README.md** file that includes:

* Your name
* What your portfolio contains
* A short description of the project

---
